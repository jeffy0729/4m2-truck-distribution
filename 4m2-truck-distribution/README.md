# 4米2轻卡城配业务分布图

全国省会及一二线城市4米2轻卡保有量与业务量数据可视化。

## 功能

- 🗺️ 地图可视化 - 全国城市分布标记
- 📊 数据展示 - 保有量、业务量统计
- 🎨 深色科技风 - 物流主题设计

## 技术栈

- Leaflet + CartoDB Dark Matter 地图
- ECharts 数据可视化
- 响应式设计

## 部署

```bash
# 本地预览
python3 -m http.server 8000
```

访问 https://jeffy0729.github.io/4m2-truck-distribution
